Install/configure the toolchainÂ¶

For configuring the built-in toolchain of custom statically built binaries, go to Edit > Preferences and select install:

The latest stable release of Apio and all its built-in packages will be installed.

Important

Some boards are not supported in the stable releases of Apio, but are available in the development branch. You might need to install/update Apio from the git repository.

NOTE: on GNU/Linux, first source ~/.icestudio/venv/bin/activate.

pip install -U git+https://github.com/FPGAwars/apio.git@develop#egg=apio
